Jasteroids!

Instructions

JasteroidS is a simple Javascript written replica of the 1979 Asteroids Atari game. Although shooting has not yet been implemented, the user is able to navigate away from the incoming/moving asteroids by using the arrow keys. Please see istructions below

1 - Click start button.

2 - Navigate the ship's direction by using the Right and Left keys.

3 - Thrust the ship by pressing the Up key.

4 - Stay alive by navigating away from the asteroids.

** If you would like to play again please click ok/hit enter twice! **
